Organize

Edit dates, locations, albums and people — right from the grid.

Select one photo or a hundred, then change their metadata in a single action. The edit menu appears in context — no separate screen, no wizard.

  • Change date, location, album, person, or event
  • Batch editing across any selection
  • Changes queued until you're ready to save
  • Writes to XMP and EXIF — compatible with any app
Fix location data

Drop a pin to correct where a photo was taken.

Photos without GPS, or with wrong coordinates, can be fixed by dragging a pin on a map. The new location is reverse-geocoded on-device and saved back to the file — no coordinates to type, no internet required.

  • Interactive pin drag — no coordinates to type
  • Reverse geocoding runs entirely on-device
  • Works on photos that never had GPS data
  • Saves GPS coordinates, country, and city
