Metadata

Your data stays portable.

Raza Photos reads from seven metadata standards and writes back to open formats — so nothing is locked in, ever. Full details →

What Raza Photos reads

  • Date taken with full timezone and format support
  • GPS coordinates, country and city
  • Titles from IPTC, XMP, or filename fallback
  • People names and face bounding boxes
  • Albums, events, and category hierarchies
  • Burst sequences from Apple MakerNote or XMP
  • File creation and modification dates
EXIF IPTC XMP GPS Apple MakerNote ACDSee XML Filesystem

What Raza Photos writes

  • Dates back to EXIF and TIFF standards
  • GPS coordinates to EXIF GPS dictionary
  • Titles to IPTC and XMP
  • Locations to IPTC, XMP, and ACDSee
  • People and face regions to XMP
  • Albums and events — compatible with Lightroom
  • Burst groupings via custom XMP namespace

Writes use CGImageDestinationCopyImageSource() — only metadata sections are touched. Your image pixels are never decoded or re-encoded.

Family controls

Let the kids browse.
Not rearrange.

Raza Photos gives you three levels of access control — no configuration wizard, no user accounts, no passwords to hand out.

🖥

Read-only server

Lock the Mac server entirely. Remote devices — every iPhone, iPad, or other Mac on the network — can browse the full library but cannot make any changes whatsoever. The library is yours to curate, from your Mac alone.

✏️

Block remote edits only

Allow browsing from any device on the network, but prevent metadata changes — no editing dates, locations, albums, or people tags from iOS. Browsing remains completely unrestricted.

📱

Read-only client

Set a specific iPhone or iPad to read-only mode on the device itself. That device becomes a pure viewer — no editing controls are shown, regardless of what the server allows.

No accounts, no passwords, no parental control apps. Just a toggle. Learn more about security →
