# Assets

Place these files here before deploying:

- `icon.png` — the app icon (1024x1024 or any square size)
- `mac.png` — Mac screenshot (windowed, with shadow)
- `iphone.png` — iPhone screenshot (in device frame)
- `ipad.png` — iPad screenshot (in device frame)
- `all-devices.png` — the composite Mac + iPad + iPhone image (used in hero)

The App Store badge is referenced inline. Replace `#app-store` in index.html with your actual App Store URL.
